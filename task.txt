Лабораторная работа №4. Сегментация изображений через поиск неоднородностей и сегментация пороговыми методами.

Оценка за лабораторную работу – 6 баллов, срок сдачи без потерь баллов – до 04.05.2022 включительно
(далее -1 балл за каждую просроченную неделю, минимум 3 балла).
Работа выполняется в группах максимум по 2 человека. Язык программирования любой,
разрешается использование сторонних библиотек. Пункты лабораторной работы считаются
зачтенными при условии корректного объяснения нижеуказанных алгоритмов.

Задание. Разработать программу, в которой необходимо реализовать сегментацию изображения:

1. через поиск неоднородностей алгоритмом выделения краев;

2. пороговым методом с глобальным порогом T, реализовать определение порога:

2.1. на основе площади (P-tile);

2.2. путем последовательных приближений;

2.3. методом k-средних (реализовать в программе сравнение при различных значениях k);

2.4. (для 2.2 и 2.3) добавить гистограмму изображения для поиска на ней пиков и последующего их сглаживания (реализовать в программе сравнение при нескольких различных количествах сглаживаний).

3. пороговым методом с адаптивным порогом (реализовать в программе сравнение при различных значениях k, величины параметра C [реализовать выбор данного параметра по окрестности через среднее, медиану и (min+max)/2] и порога T).

4. (дополнительно, +2 балла) реализовать п.1-3 для видео (для этого можно использовать OpenCV и пр. подобные библиотеки).
